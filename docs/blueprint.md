# **App Name**: Ahsan's Portfolio

## Core Features:

- Interactive Skill Bars: Visually engaging skill bars that animate to display proficiency levels.
- Dynamic Testimonial Carousel: Showcase client testimonials in an animated, auto-playing carousel.
- Animated Hero Section: Implement the animated reveal effects for text, and smooth fade-in zoom for images.
- Sticky Navigation: Implement the slide-down navigation bar which changes background on scroll.

## Style Guidelines:

- Primary color: Indigo (#4B0082) to convey expertise and professionalism.
- Background color: Light gray (#F0F0F0), offering a clean and modern look.
- Accent color: Teal (#008080), provides a refreshing and technological feel.
- Body: 'PT Sans' a humanist sans-serif for a modern feel.
- Headline: 'Space Grotesk' a sans-serif font, should the design require differentiation of text in headlines vs. body
- Use simple, flat icons, adhering to a consistent style, to represent different sections and skills.
- Employ a clean, grid-based layout to maintain order and readability, optimized for various screen sizes.
- Subtle scroll animations to progressively reveal content as the user explores the page.
